Introduction

Introduction

Hyper Angular is an angular admin template created using Angular v17 & Bootstrap v5.3.2. Hyper Angular is crafted by following the best industry standards in order to provide speed and scalability. The clean code structure allows you to build any Angular-based web application without any hustle. The theme includes beautiful dashboards, many ready-to-use hand-crafted components, widgets, pages, etc. The components can be used very easily on any page. It is fully responsive and customizable.

This documentation guide is for all users who can even be beginners or experienced angular developers.

We really care for our buyers and so in case if you are having any question or feedback, please feel free to get back to us via email support@coderthemes.com or by filling out the contact form on our website.
