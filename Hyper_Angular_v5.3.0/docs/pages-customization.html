Customization

Hyper is highly customizable and provides an easy way to change the layouts, color themes etc. Hyper comes with different layout choices, including vertical, horizontal, detached side menu, full mode etc. The layout and left sidebar related customization can be done by simply changing the value of a few variables. Everything is being controlled through rxjs. Please refer to below described details in order to see what are the possible customization or changes;

You can customize the look and feel of various elements, including navbar, topbar, form elements etc by simply editing Sass files in /src/assets/scss directory or removing unneeded components from app.scss source files.

1. How to change the default layout from vertical to another?

Open src/app/store/layout/layout-reducers.ts file, and search for initialState method. Here as a default state parameter LAYOUT, you can either specify LAYOUT_TYPES.VERTICAL or LAYOUT_TYPES.HORIZONTAL. The respective layout would get activated and being displayed as a default to user.

Hyper Default - Vertical Layout

This is already set as a default. You don't need to doanything.

Horizontal Layout

Specify default state parameter LAYOUT with LAYOUT_TYPES.HORIZONTAL

2. How to change the default sidebar layout from fluid to another?

Open src/app/store/layout/layout-reducers.ts file, and search for initialState method. Here as a default state parameter LAYOUT_MODE, you can either specify LAYOUT_MODE_TYPES.FLUID or LAYOUT_MODE_TYPES.DETACHEDor LAYOUT_MODE_TYPES.BOXED. The respective layout would get activated and being displayed as a default to user.

Detached Sidebar Layout

Specify default state parameter LAYOUT_MODE with LAYOUT_MODE_TYPES.DETACHED

Full Mode Layout

Specify default state parameter LAYOUT_MODE with LAYOUT_MODE_TYPES.FLUID

3. How to change the default color scheme from light to dark?
  • Open src/app/store/layout/layout-reducers.ts file, and search for method initialState. Here as a default LAYOUT_COLOR property of config variable, specify LAYOUT_COLOR_TYPES.DARKMODE in order to enable dark mode.

    4. How to enable boxed mode?
    1. Open src/app/store/layout/layout-reducers.ts file and search for initialState method. Here as a default LAYOUT_MODE property, specify LAYOUT_MODE_TYPES.BOXED in order to enable boxed mode.

    5. How to change the left sidebar theme?

    The left side bar (mainly available in Vertical and Detacehd layouts) can be changed very easily as well. There are three possible choices of styles:
    a. BRAND - the brand side bar with blue background
    b. Light - the sidebar with light background
    b. Dark - the sidebar with dark background

    Open src/app/store/layout/layout-reducers.ts file, and search for method initialState. Here as a default MENU_COLOR property of config variable, specify MENU_COLOR_TYPES.LIGHT or MENU_COLOR_TYPES.DARK or MENU_COLOR_TYPES.BRAND in order to activate respective sidebar theme.
