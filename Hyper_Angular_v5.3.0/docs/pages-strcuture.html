File Structure

Extract the zip file you received after purchase and you would find the "angular" folder in it with exact below files and folders: 


├── docs
├── hyper-angular
│   ├── package.json                    => Package json file
|   ├── angular.json                    => Project based configuration file
│   ├── README.md                       => Introductory documentation
│   ├── src/
│   │   └── app
│   │       └── account                 => Account module contains the implementation for login, signup, forget password, profile etc
│   │       └── component               => All common component
│   │       └── core                    => The core module - mainly contains the common services (singleton ones)
│   │       └── extrapage               => Extrapage module contains the maintenance, landing, error pages
│   │       └── layouts 
│   │           └── footer              => footer
│   │           └── horizontal          => Horizontal Topbar and top navbar, footer and right side bar
│   │           └── vertical            => Left side navbar with topbar, footer and right side bar
│   │           └── right-sidebar       => right side bar
|   |           └── shared              => contains menu json
|   |           └── private-layout      => Common container of all Layouts for private routes
│   │       └── pages  => Pages module is containing implementation of all pages including, ui kits, widgets, forms, dashboards, tables, charts, etc
|   |
│   │       └── shared                  => The shared module - containing reusable components implementation
│   │       └── store                   => The store module - containing ngrx store files
│   │       └── app.config.ts           => Main config for application
│   │       └── app.component.ts        => Entry point component
|   |       └── app.component.scss      => Styling file for Entry component
│   │       └── app.routes.ts   => Router implementation
|   |
│   │   └── assets
│   │       └── i18n                   => localization files for different language
│   │       └── scss                   => Scss files
│   │       └── fonts                  => Fonts files
│   │       └── images                 => All images
|   |
│   |   └── environments               => Contains build configuration options for particular target environments
│   │
│   ├── index.html                     => The main HTML page that is served when someone visits your site
